# Avoid The Crowd 🤐🎮

**Avoid The Crowd** is an endless runner game built using **HTML, CSS, and JavaScript only**. The goal of the game is simple — keep moving forward, avoid randomly generated obstacles, and survive as long as possible to achieve a higher score.

## 🚀 Features

* **Endless Runner Gameplay**

  * Continuous gameplay where the player moves forward endlessly.
  * Random obstacles appear throughout the game.
  * Avoid obstacles to continue playing and increase your score.

* **Multiple Control Options**

  * Control the player character using:

    * `A` key → Move Left
    * `D` key → Move Right
    * Left Arrow key → Move Left
    * Right Arrow key → Move Right
  * On-screen buttons are available for touch-based devices.

* **Responsive Gameplay**

  * The game map and obstacles automatically adjust according to the screen size.
  * Fully optimized for:

    * Desktop computers
    * Laptops
    * Tablets
    * Smartphones
  * Playable on different screen sizes and platforms without layout issues.

* **Lightweight & Fast**

  * Developed using only:

    * HTML
    * CSS
    * JavaScript
  * No external frameworks or libraries required.

## 🎯 How To Play

1. Start the game.
2. Move your character left or right to avoid incoming obstacles.
3. Keep moving forward and survive as long as possible.
4. Avoid collisions and try to achieve the highest score.

## 🛠️ Technologies Used

* **HTML5** — Game structure and elements
* **CSS3** — Styling, animations, and responsive design
* **JavaScript** — Game logic, controls, obstacle generation, scoring system
